Conditionals in JavaScript

Conditionals allow us to make decisions in our code based on certain conditions. JavaScript provides several ways to perform these checks. Let's go over the most common ones:

If Statement

The if statement checks whether a condition is true. If it is, the associated block of code is executed.


if (condition) {
    // code to execute if condition is true
}

Else If and Else

If you want to check multiple conditions, you can use else if and else to provide additional checks.


if (condition1) {
    // code to execute if condition1 is true
} else if (condition2) {
    // code to execute if condition2 is true
} else {
    // code to execute if none of the conditions are true
}

Switch Case

The switch statement is used to evaluate an expression against multiple values (cases). It’s an alternative to multiple if-else statements.


switch (expression) {
    case value1:
        // code to execute if expression === value1
        break;
    case value2:
        // code to execute if expression === value2
        break;
    default:
        // code to execute if no cases match
}

Ternary Operator

The ternary operator provides a shorthand way to write if-else statements.


let result = (condition) ? 'True' : 'False';

Example: Checking if a number is even or odd


let number = 5;
let result = (number % 2 === 0) ? 'Even' : 'Odd';
console.log(result);

Using Conditionals in Real Life

Imagine you have a website where users log in. You might want to check if a user is logged in or not:


let isLoggedIn = true;
if (isLoggedIn) {
    console.log("Welcome back!");
} else {
    console.log("Please log in.");
}
