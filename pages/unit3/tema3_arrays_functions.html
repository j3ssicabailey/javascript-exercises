Common Array Methods

push() and pop()

push() adds an element to the end of an array, while pop() removes the last element.

let fruits = ["Apple", "Banana"];
fruits.push("Cherry"); // ["Apple", "Banana", "Cherry"]
fruits.pop(); // ["Apple", "Banana"]

shift() and unshift()

shift() removes the first element, while unshift() adds an element to the beginning.

let numbers = [2, 3, 4];
numbers.unshift(1); // [1, 2, 3, 4]
numbers.shift(); // [2, 3, 4]

map()

map() creates a new array by applying a function to each element.

let numbers = [1, 2, 3];
let squared = numbers.map(num => num * num);
console.log(squared); // [1, 4, 9]

filter()

filter() creates a new array with elements that pass a condition.

let ages = [18, 21, 16, 25];
let adults = ages.filter(age => age >= 18);
console.log(adults); // [18, 21, 25]

reduce()

reduce() processes an array to a single value.

let numbers = [1, 2, 3, 4];
let sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // 10
