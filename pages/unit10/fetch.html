Using Fetch API

What is the Fetch API?

The Fetch API is a modern way to make HTTP requests in JavaScript.

Making a Simple Fetch Request


fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Handling Responses

The response from Fetch is a Response object. You can access data by calling methods like response.json(), response.text(), etc.
