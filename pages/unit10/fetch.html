Using Fetch API

What is the Fetch API?

The Fetch API is a modern interface for making HTTP requests in JavaScript. It provides a simple and flexible way to interact with resources over the web.

Making a Simple GET Request

The GET method is the default method in Fetch and is used to retrieve data from a server.


fetch('https://jsonplaceholder.typicode.com/posts/1')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error('Error:', error));

Handling Responses

The response from Fetch is a Response object. Common methods to handle response data include:

  • response.json() – Parses the response body as JSON.
  • response.text() – Reads the response as plain text.
  • response.blob() – Reads the response as a binary blob.

Making a POST Request

The POST method is used to send data to a server, typically to create a new resource.


fetch('https://jsonplaceholder.typicode.com/posts', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        title: 'New Post',
        body: 'This is the content of the new post.',
        userId: 1
    })
})
.then(response => response.json())
.then(data => console.log('Success:', data))
.catch(error => console.error('Error:', error));

Error Handling

Errors can occur when a request fails. Always use .catch() to handle them properly.


fetch('https://jsonplaceholder.typicode.com/invalid-url')
    .then(response => {
        if (!response.ok) {
            throw new Error('Network response was not ok');
        }
        return response.json();
    })
    .then(data => console.log(data))
    .catch(error => console.error('Fetch error:', error));
