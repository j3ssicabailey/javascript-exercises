Classes, Interfaces & Types

Primitive Types

  • string: Represents textual data
  • number: Represents both integers and floats
  • boolean: Represents true or false
  • null and undefined: Represent the absence of value
  • unknown: A safer alternative to any

Arrays

Arrays can be declared in multiple ways:


let numbers: number[] = [1, 2, 3];
let names: Array = ["Alice", "Bob"];

Enums


enum Color {
    Red = "RED",
    Green = "GREEN",
    Blue = "BLUE"
}
let myColor: Color = Color.Red;

Avoid Using any!

Using any removes type safety and should be avoided unless absolutely necessary.


let value: any = "Hello"; // Avoid this when possible
