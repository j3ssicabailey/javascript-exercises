Union vs Intersection

Union Types

A union type allows a variable to hold multiple types.


let value: string | number;
value = "Hello"; // Valid
value = 42; // Valid
value = true; // ‚ùå Error: not a string or number

Intersection Types

An intersection type combines multiple types into one.


interface Person {
    name: string;
    age: number;
}
interface Employee {
    role: string;
}
type Worker = Person & Employee;
let worker: Worker = { name: "Alice", age: 30, role: "Developer" };

When to Use

  • Use Union when a value can be one of multiple types.
  • Use Intersection when combining multiple type requirements.
