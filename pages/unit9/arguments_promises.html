Using Arguments with Promises

Passing Arguments to Promises

Promises can take arguments, which can be used inside the executor function to determine the outcome of the promise.

Example with a Function Parameter


const checkNumber = (num) => {
    return new Promise((resolve, reject) => {
        if (num > 10) {
            resolve(`${num} is greater than 10`);
        } else {
            reject(`${num} is not greater than 10`);
        }
    });
};

checkNumber(15)
    .then(result => console.log(result))
    .catch(error => console.error(error));

Turning Promises into Arrow Functions

Promises can be written more concisely using arrow functions.

Standard Function vs Arrow Function


// Traditional function
function fetchData(url) {
    return new Promise((resolve, reject) => {
        fetch(url)
            .then(response => response.json())
            .then(data => resolve(data))
            .catch(error => reject(error));
    });
}

// Arrow function version
const fetchDataArrow = (url) => new Promise((resolve, reject) => {
    fetch(url)
        .then(response => response.json())
        .then(resolve)
        .catch(reject);
});

Example with Multiple Arguments


const multiplyNumbers = (a, b) => new Promise((resolve, reject) => {
    if (typeof a === "number" && typeof b === "number") {
        resolve(a * b);
    } else {
        reject("Both arguments must be numbers");
    }
});

multiplyNumbers(4, 5)
    .then(result => console.log("Result:", result))
    .catch(error => console.error("Error:", error));
