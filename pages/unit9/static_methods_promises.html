Static Methods in Promises

Static Methods Overview

JavaScript provides several static methods for handling multiple promises efficiently.

Promise.all()

Waits for all promises to resolve or rejects if any fail.


const promise1 = Promise.resolve(10);
const promise2 = new Promise(resolve => setTimeout(() => resolve(20), 1000));
const promise3 = new Promise((_, reject) => setTimeout(() => reject("Error"), 500));

Promise.all([promise1, promise2, promise3])
    .then(values => console.log(values))
    .catch(error => console.error(error)); // Rejects immediately with "Error"

Promise.race()

Resolves or rejects as soon as the first promise settles.


const fast = new Promise(resolve => setTimeout(() => resolve("Fast!"), 500));
const slow = new Promise(resolve => setTimeout(() => resolve("Slow"), 2000));

Promise.race([fast, slow]).then(result => console.log(result)); // "Fast!" after 500ms

Promise.any()

Resolves with the first fulfilled promise or rejects if all fail.


const p1 = new Promise((_, reject) => setTimeout(() => reject("Fail 1"), 500));
const p2 = new Promise(resolve => setTimeout(() => resolve("Success!"), 1000));
const p3 = new Promise((_, reject) => setTimeout(() => reject("Fail 3"), 1500));

Promise.any([p1, p2, p3])
    .then(value => console.log(value)) // "Success!" after 1000ms
    .catch(error => console.error(error));

Promise.allSettled()

Waits for all promises and returns their results, regardless of success or failure.


const promiseA = Promise.resolve("Resolved A");
const promiseB = Promise.reject("Rejected B");
const promiseC = new Promise(resolve => setTimeout(() => resolve("Resolved C"), 1000));

Promise.allSettled([promiseA, promiseB, promiseC])
    .then(results => console.log(results));
