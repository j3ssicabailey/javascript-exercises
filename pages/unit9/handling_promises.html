Handling Promises

Once a promise is created, we handle its result using the methods .then(), .catch(), and .finally().

The .then() Method

The .then() method is used to handle the resolved value of a promise.


let myPromise = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Promise fulfilled!"), 2000);
});

myPromise.then(result => {
    console.log(result); // "Promise fulfilled!"
});

The .catch() Method

The .catch() method handles any rejection from the promise.


let myPromise = new Promise((resolve, reject) => {
    setTimeout(() => reject("Something went wrong!"), 2000);
});

myPromise.catch(error => {
    console.log(error); // "Something went wrong!"
});

The .finally() Method

The .finally() method runs regardless of whether the promise was resolved or rejected.


let myPromise = new Promise((resolve, reject) => {
    setTimeout(() => resolve("Done!"), 2000);
});

myPromise
    .then(result => console.log(result))
    .catch(error => console.log(error))
    .finally(() => console.log("Promise finished"));
