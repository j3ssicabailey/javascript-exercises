Promises in JavaScript

What Are Promises?

A promise in JavaScript represents a value that may be available now, in the future, or never. It helps handle asynchronous operations efficiently.

Creating a Promise

A Promise is created using the new Promise constructor, which takes a function with two parameters: resolve (for success) and reject (for failure).


const myPromise = new Promise((resolve, reject) => {
    let success = true;
    if (success) {
        resolve("Operation successful");
    } else {
        reject("Operation failed");
    }
});

Promise States

  • Pending: Initial state, before the promise is resolved or rejected.
  • Fulfilled: The operation was successful.
  • Rejected: The operation failed.
