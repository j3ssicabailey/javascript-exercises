Understanding Classes in JavaScript

JavaScript introduced classes in ES6 to make working with objects and prototypes easier. A class is essentially a blueprint for creating objects.

Defining a Class

We define a class using the class keyword. Inside the class, we can define a constructor method, which initializes object properties.

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    introduce() {
        console.log(`Hi, I'm ${this.name} and I'm ${this.age} years old.`);
    }
}

const alice = new Person("Alice", 25);
alice.introduce(); // "Hi, I'm Alice and I'm 25 years old."

Class Inheritance

Classes can inherit from other classes using the extends keyword. This allows us to create specialized versions of a base class.

class Student extends Person {
    constructor(name, age, subject) {
        super(name, age);
        this.subject = subject;
    }

    study() {
        console.log(`${this.name} is studying ${this.subject}.`);
    }
}

const bob = new Student("Bob", 22, "JavaScript");
bob.introduce(); // "Hi, I'm Bob and I'm 22 years old."
bob.study(); // "Bob is studying JavaScript."

Getters and Setters

We can use get and set methods to control access to properties.

class Car {
    constructor(brand, speed) {
        this.brand = brand;
        this._speed = speed;
    }

    get speed() {
        return `${this._speed} km/h`;
    }

    set speed(value) {
        if (value > 0) {
            this._speed = value;
        } else {
            console.log("Speed must be positive.");
        }
    }
}

const myCar = new Car("Toyota", 120);
console.log(myCar.speed); // "120 km/h"
myCar.speed = 150;
console.log(myCar.speed); // "150 km/h"
