1. Function Declarations

A function declaration defines a function using the function keyword. It can be called before its definition due to hoisting.

function greet(name) {
    return "Hello, " + name + "!";
}

console.log(greet("Jess")); // Output: Hello, Jess!

2. Function Expressions

Functions can also be stored in variables. Unlike declarations, function expressions are not hoisted.

const greet = function(name) {
    return "Hello, " + name + "!";
};

console.log(greet("Jess")); // Output: Hello, Jess!

3. Arrow Functions (Lambda Functions)

Arrow functions provide a shorter syntax for writing functions.

// Standard function
const add = function(a, b) {
    return a + b;
};

// Arrow function equivalent
const add = (a, b) => a + b;

console.log(add(3, 5)); // Output: 8

4. Immediately Invoked Function Expressions (IIFE)

An IIFE (Immediately Invoked Function Expression) is a function that runs immediately after being defined.

(function() {
    console.log("This function runs immediately!");
})();
// Output: This function runs immediately!

5. Closures

A closure is a function that remembers the variables from its outer scope even after the outer function has finished execution.

function counter() {
    let count = 0;
    return function() {
        count++;
        return count;
    };
}

const increment = counter();
console.log(increment()); // Output: 1
console.log(increment()); // Output: 2
