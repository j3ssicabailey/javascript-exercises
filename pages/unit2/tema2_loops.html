Loops in JavaScript

Loops allow you to repeat a block of code multiple times. They are useful when you need to perform repetitive tasks. JavaScript provides three common types of loops:

For Loop

The for loop is used when you know the number of times you want to repeat a block of code.

for (let i = 0; i < 5; i++) {
    console.log(i); // prints 0, 1, 2, 3, 4
}

While Loop

The while loop repeats a block of code while a condition is true. It's useful when you don't know the number of iterations ahead of time.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Do While Loop

The do while loop is similar to the while loop, but it guarantees the block of code will run at least once, even if the condition is false initially.

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

When to Use Each Loop

Use the for loop when you know how many times you want to repeat the code. Use the while loop when you don't know how many iterations you'll need, and use the do while loop when you need to ensure the code runs at least once.

Example: Summing Numbers

Let’s say you want to calculate the sum of numbers from 1 to 5:

let sum = 0;
for (let i = 1; i <= 5; i++) {
    sum += i; // add i to sum
}
console.log(sum); // 15

Nested Loops

You can also use loops inside of other loops, known as nested loops.

for (let i = 1; i <= 3; i++) {
    for (let j = 1; j <= 3; j++) {
        console.log(`i = ${i}, j = ${j}`);
    }
}

Breaking Out of Loops

You can break out of a loop early using the break statement:

for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i);
}

Skipping an Iteration

You can skip the current iteration and move to the next one using the continue statement:

for (let i = 0; i < 5; i++) {
    if (i === 3) continue; // skips the rest of the code when i is 3
    console.log(i);
}
