Types of Events in JavaScript

Form Events

These events occur when interacting with forms.

  • submit: Triggered when a form is submitted.
  • change: Occurs when the value of an input changes.
  • focus: Fired when an element gains focus.
  • blur: Fired when an element loses focus.

document.querySelector("form").addEventListener("submit", function(event) {
    event.preventDefault();
    console.log("Form submitted!");
});

document.querySelector("input").addEventListener("change", function() {
    console.log("Input value changed to: " + this.value);
});

document.querySelector("input").addEventListener("focus", function() {
    console.log("Input focused");
});

document.querySelector("input").addEventListener("blur", function() {
    console.log("Input lost focus");
});

Mouse Events

These events are triggered by mouse actions.

  • click: Fired when an element is clicked.
  • dblclick: Triggered on a double click.
  • mousemove: Fired when the mouse moves over an element.
  • mouseover: Triggered when the mouse enters an element.
  • mouseout: Triggered when the mouse leaves an element.

document.querySelector("button").addEventListener("click", function() {
    console.log("Button clicked!");
});

document.querySelector("button").addEventListener("dblclick", function() {
    console.log("Button double-clicked!");
});

document.querySelector("div").addEventListener("mousemove", function(event) {
    console.log(`Mouse position: X=${event.clientX}, Y=${event.clientY}`);
});

Keyboard Events

These events are triggered by keyboard interactions.

  • keydown: Fired when a key is pressed.
  • keyup: Fired when a key is released.
  • keypress: (Deprecated) Fired when a key is pressed.

document.addEventListener("keydown", function(event) {
    console.log("Key down: " + event.key);
});

document.addEventListener("keyup", function(event) {
    console.log("Key up: " + event.key);
});

Drag and Drop Events

These events enable drag-and-drop functionality.

  • dragstart: Fired when an element starts being dragged.
  • dragover: Fired when an element is dragged over a valid drop target.
  • drop: Fired when an element is dropped.

document.querySelector("#draggable").addEventListener("dragstart", function(event) {
    event.dataTransfer.setData("text", event.target.id);
});

document.querySelector("#dropzone").addEventListener("dragover", function(event) {
    event.preventDefault();
});

document.querySelector("#dropzone").addEventListener("drop", function(event) {
    event.preventDefault();
    let data = event.dataTransfer.getData("text");
    event.target.appendChild(document.getElementById(data));
});

Multimedia Events

These events occur when working with multimedia elements.

  • play: Fired when a video or audio starts playing.
  • pause: Fired when playback is paused.
  • ended: Fired when playback reaches the end.

document.querySelector("video").addEventListener("play", function() {
    console.log("Video is playing");
});

document.querySelector("video").addEventListener("pause", function() {
    console.log("Video is paused");
});

document.querySelector("video").addEventListener("ended", function() {
    console.log("Video has ended");
});

Other Events

  • DOMContentLoaded: Fired when the HTML document has been fully loaded and parsed.
  • resize: Triggered when the window is resized.
  • scroll: Fired when the page is scrolled.

document.addEventListener("DOMContentLoaded", function() {
    console.log("DOM fully loaded and parsed");
});

window.addEventListener("resize", function() {
    console.log("Window resized to: " + window.innerWidth + "x" + window.innerHeight);
});

window.addEventListener("scroll", function() {
    console.log("User scrolled the page");
});
