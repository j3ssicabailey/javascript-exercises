Selectors in JavaScript

Selecting elements in the DOM is the first step to manipulating them. JavaScript provides multiple methods to achieve this.

Common Selectors

  • getElementById: Selects an element by its ID.
  • getElementsByClassName: Selects elements by class name.
  • getElementsByTagName: Selects elements by tag name.
  • getElementByName: Selects elements by the name attribute.
  • querySelector: Selects the first element that matches a CSS selector.
  • querySelectorAll: Selects all elements that match a CSS selector.

HTMLCollection vs NodeList

Understanding the difference between these two data structures is important for working efficiently with the DOM.

  • HTMLCollection: A live collection of elements, meaning changes in the DOM update the collection automatically.
  • NodeList: A static or live collection of nodes depending on how it is obtained. querySelectorAll returns a static NodeList.

Scoped Searches

Instead of searching the entire document, you can narrow the search within a specific element.


const container = document.getElementById("container");
const items = container.getElementsByClassName("item");

Here, only elements with class "item" inside the element with ID "container" will be selected.

Example: Selecting an Element by ID


document.getElementById("example").style.color = "red";

This example selects an element with the ID "example" and changes its text color to red.
