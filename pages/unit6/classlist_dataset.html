classList and dataset in JavaScript

Managing CSS Classes with classList

The classList property provides an easy way to manage the class attribute of an element.

Common Methods

  • element.classList.add("new-class") - Adds a class.
  • element.classList.remove("old-class") - Removes a class.
  • element.classList.toggle("toggle-class") - Toggles a class on/off.
  • element.classList.contains("class-name") - Checks if a class exists.
  • element.classList.replace("old", "new") - Replaces one class with another.

Example: Toggling a Class


const element = document.getElementById("myDiv");
element.classList.toggle("highlight");

This will add highlight if it's not there, or remove it if it is.

Working with dataset

The dataset property allows access to data-* attributes of an element.

Accessing Data Attributes

  • element.dataset.key - Retrieves the value of data-key.
  • element.dataset.key = "new value" - Modifies a dataset attribute.
  • delete element.dataset.key - Removes a dataset attribute.

Example: Using dataset


const button = document.getElementById("myButton");
console.log(button.dataset.action); // Outputs: "save"
button.dataset.action = "delete";

If <button id="myButton" data-action="save"> exists, this modifies its data-action value.
