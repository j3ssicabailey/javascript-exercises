Intro: Creating Elements in the DOM

JavaScript provides multiple methods to dynamically create and manipulate elements in the DOM.

Common Methods

  • createElement: Creates a new HTML element.
  • createTextNode: Creates a text node (useful for inserting raw text).
  • createComment: Creates a comment node.
  • appendChild: Appends a node as the last child of a parent.
  • insertBefore: Inserts a node before a specified child.
  • removeChild: Removes a child node from a parent.
  • replaceChild: Replaces an existing child node with a new node.
  • cloneNode: Duplicates a node (deep cloning copies child nodes as well).
  • createDocumentFragment: Creates a lightweight document fragment for efficient DOM updates.

Node Properties

  • isConnected: Checks if a node is in the document.
  • textContent: Gets or sets the text content of an element (ignores HTML structure).
  • innerHTML: Gets or sets the HTML content of an element (parses HTML).

Example: Creating and Appending an Element


const newDiv = document.createElement("div");
newDiv.textContent = "Hello, world!";
document.body.appendChild(newDiv);

Example: Using createDocumentFragment


const fragment = document.createDocumentFragment();
for (let i = 0; i < 3; i++) {
    let p = document.createElement("p");
    p.textContent = `Paragraph ${i + 1}`;
    fragment.appendChild(p);
}
document.body.appendChild(fragment);
