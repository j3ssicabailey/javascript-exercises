Managing Attributes in the DOM

Accessing Basic Attributes

Some attributes, like id, class, and src, can be accessed directly as properties.


const img = document.getElementById("myImage");
console.log(img.src); // Access the src attribute
console.log(img.id);  // Access the id attribute

Accessing Other Attributes

For attributes that do not have a direct property, use getAttribute and hasAttribute.


const link = document.querySelector("a");
console.log(link.getAttribute("href")); // Gets the href attribute
console.log(link.hasAttribute("target")); // Checks if target exists

Modifying Attributes

  • setAttribute: Adds or updates an attribute.
  • removeAttribute: Removes an attribute.
  • toggleAttribute: Toggles an attribute (adds it if missing, removes it if present).

const button = document.querySelector("button");
button.setAttribute("disabled", "true"); // Adds disabled attribute
button.removeAttribute("disabled"); // Removes disabled attribute
button.toggleAttribute("data-active"); // Toggles a custom attribute

Boolean Attributes

Boolean attributes like checked, disabled, and readonly should be managed as properties.


const checkbox = document.querySelector("input[type='checkbox']");
console.log(checkbox.checked); // true or false
checkbox.checked = true; // Sets the checked state
