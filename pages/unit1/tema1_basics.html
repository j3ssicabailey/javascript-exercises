Ways to Use JavaScript

  • Inline: <button onclick="alert('Hello!')">Click me</button>
  • Internal: Using <script> inside HTML
  • External: Linking a .js file
  • Console: Running JavaScript in browser dev tools

Variables

JavaScript has three types of variables:

  • var - Function-scoped, can be reassigned
  • let - Block-scoped, can be reassigned
  • const - Block-scoped, cannot be reassigned

Data Types

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Object
  • Symbol
  • BigInt

Operators

  • Arithmetic: + - * / % **
  • Comparison: == === != !== > < >= <=
  • Logical: && || !
  • Assignment: = += -= *= /=

Functions

Functions allow code reuse:

function greet(name) {
    return `Hello, ${name}!`;
}
console.log(greet("Jess"));
